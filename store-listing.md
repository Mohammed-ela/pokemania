## Description complète

Pokemania est un Pokédex complet et gratuit qui te permet d'explorer l'univers des Pokémon directement depuis ton smartphone.

FONCTIONNALITÉS PRINCIPALES

- Pokédex complet : Parcours les 1025 Pokémon de toutes les générations dans une interface fluide et moderne.
- Recherche avancée : Filtre par nom, type ou génération pour trouver rapidement n'importe quel Pokémon.
- Statistiques détaillées : Consulte les stats de base (PV, Attaque, Défense, etc.), le BST, les talents et les talents cachés de chaque Pokémon.
- Chaînes d'évolution : Visualise les évolutions complètes avec navigation interactive.
- Matchups de types : Découvre les faiblesses, résistances et immunités de chaque Pokémon.
- Données compétitives : Groupes d'œufs, taux de capture, ratio de genre et caractéristiques physiques.
- Favoris : Sauvegarde tes Pokémon préférés pour y accéder rapidement.
- Mode hors ligne : Les données consultées sont mises en cache pendant 7 jours pour un accès sans connexion.
- Mode sombre : Bascule entre le thème clair et sombre selon tes préférences.
- Formes spéciales : Découvre les variantes Gigamax et Shiny.

CONÇUE POUR TA VIE PRIVÉE

Pokemania ne collecte aucune donnée personnelle. Pas de compte à créer, pas de publicité, pas de tracking. Tes favoris restent uniquement sur ton appareil. L'application nécessite seulement une connexion Internet pour charger les données Pokémon.

APPLICATION 100% GRATUITE

Aucune publicité, aucun achat intégré, aucune fonctionnalité premium cachée. Toutes les fonctionnalités sont accessibles gratuitement.

Données fournies par l'API Tyradex (https://tyradex.app).

## Captures d'écran requises

Google Play exige au minimum 2 captures d'écran (format téléphone).
Recommandation : 4 à 8 captures de ces écrans :

1. Écran d'accueil (HomeScreen) - Montre le branding et la navigation
2. Liste des Pokémon (PokemonListScreen) - Grille avec la recherche
3. Détail d'un Pokémon populaire (ex: Pikachu ou Dracaufeu) - Stats de base
4. Détail - Section évolutions
5. Détail - Matchups de types
6. Écran de recherche avancée avec filtres
7. Écran des favoris avec quelques Pokémon sauvegardés
8. Mode sombre (n'importe quel écran)

Format requis : JPEG ou PNG 24 bits, entre 320px et 3840px, ratio 16:9 ou 9:16
Taille recommandée : 1080 x 1920 px (portrait)
